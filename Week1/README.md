# Week 1 - Banking System Project

## 🎯 Week 1 Objectives
- ✅ Hello World PHP Test
- ✅ Basic MySQL PDO Connectivity
- ✅ Database Schema Design
- ✅ Initial Setup & Configuration

## 📁 Files Included
- `index.php` - Hello World test page
- `db_test.php` - Database connectivity verification
- `week1_db.sql` - Database schema and sample data

## 🚀 Quick Start
1. Import `week1_db.sql` into phpMyAdmin or MySQL CLI:
   ```bash
   mysql -u root < week1_db.sql
   ```

2. Open in browser:
   - Hello World: http://localhost/banking_system/Week1/index.php
   - DB Test: http://localhost/banking_system/Week1/db_test.php

## 📊 Database Schema
- **Users Table**: Stores user credentials and balance
- **Transactions Table**: Records all deposits/withdrawals

## 🔧 Configuration
Update connection string in `db_test.php` if needed:
- Host: 127.0.0.1
- Database: banking_system
- User: root
- Password: (empty)
