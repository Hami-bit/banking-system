# Week 4 - Banking System

This folder contains a simple banking dashboard with PHP session authentication and transaction management.

## Features
- User login with PHP session.
- Protected API endpoints for authenticated users.
- Balance display and deposit/withdraw actions.
- Transaction feed with recent activity.

## Files
- `login.php` — login form and authentication.
- `logout.php` — session logout.
- `dashboard.html` — banking dashboard page.
- `dashboard.js` — client-side API calls.
- `api.php` — authenticated transaction API.
- `style.css` — dashboard styling.

## Run
1. Place this folder under XAMPP `htdocs/banking_system/Week4/`.
2. Open `http://localhost/banking_system/Week4/login.php`.
3. Use the database from `Week1/week1_db.sql` and create users manually for login.
